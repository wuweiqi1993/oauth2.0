#oauth2.0授权框架
##简介
Ouath2.0授权框架支持第三方应用通过协调资源所属者与HTTP服务之间的交互（交流）来代表资源所属者去获取一个HTTP服务的受限资源。或者允许第三方以自己的名义来访问受限资源。这份框架说明替换并淘汰了在RFC 5849内描述的Oauth1.0协议（过时）。
